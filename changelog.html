SeedOverlay

  • Added SeedOverlay module
  • Radius is the chunk radius around the player which gets scanned
  • Interval is the interval between scans in ticks
  • IgnoreCommon ignores common blocks
  • IgnoreGrass ignores grass that’s grown on dirt block
  • Note: This will only work in default/vanilla 1.12.2 chunks

ClickGUI

  • Added toggle option for the category bar
  • Added Mercury bar mode
  • Added HideBar - automatically hides the category bar when you’re not hovering over it

TickShift

  • Added TickShift module
  • Speed is the tickrate multiplier during a burst
  • Ticks is the amount of ticks a tick shift burst lasts
  • MinInterval is the minimum interval in seconds in between tickshift bursts
  • Exploit extends the time you can burst for, but might not work on some servers
    • StartDelay is the amount of seconds to exploit after enabling the module
    • Repeat applies the start delay after every burst
    • Frequency is the frequency of the exploit after the start delay - 0 will disable the exploit entirely after the start delay passes
  • Delayed delays sending packets during bursts

Speed

  • Improved reliability of StrafeStrict
  • Fixed StrafeStrict and BHop rubberbanding in 2 block high areas
  • Added ResistSpeed option - this tries to resist the Swiftness effect if you have it, but this might not work on every server

AutoXP

    • AvoidCrystals will not mend while there are crystals nearby
